隐式类型转换

  • isNaN ( )

    判断这个值是不是数据类型NaN，

    判断过程：*先将括号里面的参数通过Number（）转换，之后的值再判断。

    isNaN ( ‘abc’ ) —-> Number ( ‘abc’ ) = NaN —-> true`

​ isNaN ( null ) —-> Number ( null ) = 0 —-> false`

  • 一元正负 （ ++ / – + / - ）

    只要这些符号在那儿一立，通通先过一次Number函数，转成数字类型再进行加加和减减，如果不能转换成数字，就是NaN，不能转成数字也是数字类型的人了。

    num = ‘ 123 ’ ---> num == Number( '123' ) == 123 ---> num ++ ---> 124

    num = ' abc ' ---> num == Number( 'abc' ) == NaN ---> num ++ ---> NaN

  • 加号 （ + ）

    加号的特殊点在于，只要加号两侧有一个字符串，就会调用 * String（）

  • 减、乘、除、求模

    同样先调用 Number（）

    其余同理。

  • 逻辑与 &&、逻辑或 ||、非 ！

    数值转换成boolean 的规则 :

    1. 对象、非零整数、非空字符串返回true
    2. 空字符串、null、undefined、NaN、false、0 ,返回false

    • 在条件判断中：

      逻辑与 &&： 所有条件都为真，整体为真 返回true 否则返回false

      逻辑或 ||： 所有条件都为假，整体才为假返回false 否则返回true

    • 在赋值操作中：

      var value = A || B，只看A的值是真还是假，如果为真返回A的值，否则返回B的值。

      var value = A && B ，只看A的值，A为假返回A的值，否则返回B的值。

  • < > <= >=

    操作数只要有一个是数字类型的，另外一个就通过Number（）转换为数字再比较。

    如果两个操作数都是字符串，则对比ASCII值。

    其他同理

  • == !=

    同样有隐式类型转换，先转number再比较

    1 > 2 false

    1 > '2' ---> 1 > Number( '2' ) ---> 1 > 2 false

    false > true ---> Number( false ) > Number( true ) ---> 0 > 1 false

​ null > 0 false`

​ undefined == null true`

不发生数据类型转换的

1 !== 1 false

1 !== '1' true
